HealthFi Vault - Neon Frontend (UI-only, mock backend)

This project is a Create React App style UI prototype for HealthFi Vault with a dark neon theme.
It uses localStorage to mock user auth and vault data so you can test the full flow without a backend.

Run locally:
1. npm install
2. npm start

Pages included:
- Landing
- Home
- Login
- Signup
- Dashboard (mock vaults)
- Create Vault (adds vaults to localStorage)
- Settings (edit profile, change password, delete account)

Note: This is mock frontend only; backend and smart contract integration are not included.
